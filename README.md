# 📊 Amazon Sales Data Analysis using Power BI

## 📝 Project Overview

This project focuses on analyzing Amazon sales data using Power BI to extract meaningful business insights. The raw data is transformed into interactive dashboards that help in understanding sales performance, customer behavior, and product trends.

The project demonstrates how data visualization can support better decision-making in e-commerce businesses.

---

## 🎯 Objectives

* Analyze sales performance and revenue trends
* Identify top-performing products and categories
* Understand regional sales distribution
* Gain insights into customer purchasing behavior
* Build interactive dashboards for decision-making

---

## 🛠️ Tools & Technologies

* **Power BI** – Dashboard creation and visualization
* **Microsoft Excel / CSV** – Data source
* **Data Cleaning Techniques** – Preprocessing and transformation

---

## 📂 Dataset Information

The dataset contains Amazon sales records including:

* Order ID
* Product Category
* Sales Amount
* Profit
* Quantity Sold
* Order Date
* Region

---

## 🔄 Project Workflow

### 1️⃣ Data Collection

* Imported dataset from Excel/CSV into Power BI

### 2️⃣ Data Cleaning

* Removed duplicate records
* Handled missing values
* Converted data types (date, currency, etc.)

### 3️⃣ Data Transformation

* Created calculated columns and measures
* Organized data for better analysis

### 4️⃣ Dashboard Development

* Built interactive dashboards using charts and KPIs
* Added slicers and filters for user interaction

### 5️⃣ Insights Generation

* Identified sales trends and patterns
* Found top-performing products and regions

---

## 📊 Dashboard Features

### 🔹 Sales Overview Dashboard

* Total Sales
* Total Profit
* Total Orders

---

## 📈 Key Insights

* Certain product categories generate higher revenue
* Seasonal trends impact sales performance
* Some regions contribute more to overall profit
* Identified areas with low profitability

---

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open it using Power BI Desktop
3. Explore dashboards using filters and slicers
